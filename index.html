JCAMP File Converter

It is a simple application that converts image metadata in JCAMP format to CSV format.

1. Select A Folder

Choose a folder that contains JCAMP files. Once this step is done, step 2 will be available.

Progress

0% done.
0% Complete (success)

2. Generate CSV File - disabled until step 1 is completed.

Click on the Generate button to generate a file. Then the newly generated file will be placed in the Result Files section. The file can then be opened, viewed, downloaded, and deleted. Generated files are kept until they are manually deleted or Chrome browser refreshes its cache.
